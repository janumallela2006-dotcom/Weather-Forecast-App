# Weather Forecast App – Project Description

The **Weather Forecast App** is a responsive web application developed using **HTML, CSS, and JavaScript**. It enables users to search for any city and view real-time weather information. The application fetches live weather data from the **Open-Meteo API** , making it simple, efficient, and easy to use.

The app provides important weather details such as **temperature, humidity, wind speed, city name, and country** through a clean and user-friendly interface. It also includes input validation, error handling for invalid city names, and a responsive design that works on desktops, tablets, and mobile devices.

**Technologies Used:**

* HTML5
* CSS3
* JavaScript (ES6)
* Open-Meteo Geocoding API
* Open-Meteo Weather API

**Key Features:**

* Search weather by city name
* Display real-time temperature
* Show humidity and wind speed
* Display city and country information
* Responsive and modern user interface
* Input validation and error handling
